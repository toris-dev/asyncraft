# Code of Conduct

`asyncraft` follows a simple standard to keep discussions and contributions professional and welcoming.

## Our Standards

All participants are expected to:

- Be respectful and considerate in comments, reviews, issues, and pull requests.
- Focus criticism on code and ideas, not individuals.
- Use clear, concise language and avoid harassment, hate speech, personal attacks, or discriminatory remarks.
- Respect different skill levels and be patient when helping newcomers.
- Report behavior that feels unsafe, abusive, or hostile.

## Unacceptable Behavior

The following are not acceptable in this project:

- Personal insults, threats, intimidation, or targeted harassment.
- Discriminatory comments by identity, ability, experience, or background.
- Repeatedly derailing discussions or making low-effort, abusive remarks.
- Sharing private information without consent.
- Any conduct intended to disrupt collaboration.

## Reporting

If you witness or experience conduct that violates this Code of Conduct, report it to the maintainer:

- GitHub: create an issue titled "Code of Conduct" and request maintainer follow-up.
- Email: `ironjustlikethat@gmail.com` (with a concise description and relevant links).

Reports are handled with confidentiality and prompt review.

## Scope

This policy applies to:

- All GitHub issue threads and pull requests in this repository.
- Any project communication channels connected to `asyncraft`.

Thanks for helping keep `asyncraft` a safe and constructive place to build software.
