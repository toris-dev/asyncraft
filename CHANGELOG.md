## [0.1.1] - 2026-07-24

### Added

- Added comprehensive open-source governance and contribution documentation: `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`, `SUPPORT.md`, `MAINTAINERS.md`, `ROADMAP.md`, `RELEASING.md`, FAQ, and examples documentation.
- Added GitHub community and maintenance templates for issues, discussions, and PRs.
- Added funding and dependency update configuration for GitHub-hosted open-source hygiene.

### Changed

- Improved package metadata, release scripts, file package list, and README badges/links for public visibility.
- Added CI and Release GitHub Workflows for automated checks and publication flow.

### Fixed

- Added initial open-source readiness process for contribution, review, and support flow consistency.

### Security

- Added security reporting guidance for vulnerabilities via security advisories and email.

## [0.1.0] - 2026-07-24

### Added

- Initial release with core async primitives: `retry`, `withTimeout`, `createLimit`, `asyncMap`, and `sleep`.
- `AbortSignal` integration for cancellable timeout/retry/sleep flows.
- ESM + CJS builds with strict TypeScript types.
- Core unit tests using Vitest.
