# Security Policy

If you discover a security issue in asyncraft, please report it privately.

## Reporting

- Open a [GitHub Security Advisory](https://github.com/toris-dev/asyncraft/security/advisories/new) if you have a GitHub account.
- If you cannot use advisories, email `ironjustlikethat@gmail.com`.

Please include:

- Affected version(s)
- Steps to reproduce
- Impact assessment
- Proof-of-concept or affected code sample if possible

We do our best to acknowledge reports within 5 business days and provide a fix timeline based on severity.

## Supported versions

We support only the latest published release in the `main` branch for security fixes.

## Safe disclosure

Please do not post potential vulnerabilities on public issue trackers before a fix is released.
